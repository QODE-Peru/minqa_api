    Como usar el API

    Generar el token

    • Petición: 
    Get
    • Url: 
    http://192.168.65.91:4000/api/token
    • Header: 
    Content-type:application/json
    • Body: 
    name:minqa
password:123456789
    • Respuesta: 
    {
    "jwt": "el token generado va aqui",
    "exp": 1529785116
}

    Obtener cuentas bancarias

    • Petición: 
    Post
    • Url: 
    http://192.168.65.91:4000/api/account
    • Header: 
    Content-type:application/json
Authorization:Bearer <token>
    • Body: 
    card_number:1234123412341234
password:123456
    • Respuesta: 
    {
    "cuentas": [
        {
            "saldo": 300,
            "numero": "123-1234567",
            "id": 1
        }
    ],
    "card_number": 1234123412341234
}
